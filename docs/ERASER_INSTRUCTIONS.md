# Eraser.io Diagram Instructions

## Files Created

1. **architecture-diagram.eraser** - Complete system architecture
2. **dataflow-diagram.eraser** - Activity categorization flow

## How to Use

### Step 1: Open Eraser.io
1. Go to https://app.eraser.io/
2. Create a new diagram or open existing workspace

### Step 2: Import Diagram Code
1. Click on "Code" or "Diagram as Code" mode
2. Copy the contents of `architecture-diagram.eraser`
3. Paste into the eraser.io editor
4. The diagram will auto-generate

### Step 3: Customize
- Adjust layout by dragging components
- Change colors in the code: `color: blue|green|orange|purple|red|yellow`
- Modify icons: `icon: monitor|server|database|brain|shield|bell|chart-bar`
- Add more notes or annotations as needed

### Step 4: Export
1. Click "Export" in eraser.io
2. Choose format: PNG, SVG, or PDF
3. Use high resolution for presentation
